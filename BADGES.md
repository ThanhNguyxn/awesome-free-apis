## üé® Alternative Styles

Shields.io supports different styles. Replace `?style=` parameter:

- `?style=flat` - Default flat style
- `?style=flat-square` - Flat with square corners
- `?style=plastic` - Plastic style
- `?style=for-the-badge` - Large badges
- `?style=social` - GitHub style (for stars/forks)

Example:
```markdown
[![Stars](https://img.shields.io/github/stars/ThanhNguyxn/awesome-free-apis?style=for-the-badge)](https://github.com/ThanhNguyxn/awesome-free-apis)
```

## üìù Usage Tips

3. **Don't overdo it** - 6-8 badges is ideal
4. **Update regularly** - Badges auto-update from GitHub
