### Quick Start

```bash
# Clone the repo
git clone https://github.com/ThanhNguyxn/awesome-free-apis.git
cd awesome-free-apis/examples

# Run a Python example
python pokemon/pokemon_example.py

# Run a JavaScript example
node pokemon/pokemon_example.js
```
